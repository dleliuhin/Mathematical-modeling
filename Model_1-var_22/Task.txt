Задачу решаем 3 раза для разных a=1,2,4
Нужно построить зависимость стационарных решений от tau:
x2(tau), x1(tau)

Фактически tau это p2 или p5
По сути p2 и p5 это неизвестные

Построить диаграмму решений

Стационарные решения - обе производные равны 0

неизветных получается 3 шт - это два икса и tau

в уравнении 2 переменных три

можно брать по tau

решить 2 уравнения с двумя неизвестными

x2 хуже всего входит

=) цикл по x2
а два других уравнения решаем 

1) 1 уравнение умножаем на p4  - вычтем из нижнего
  x1 - влево
  
2) подставляем в 1-ое уравнение

Диапазон значений x2: -5 до 10

Прграмма выглядит так:

цикл по x2
решаем квадратное уравнение
находим два корня
для каждого корня по tau находим x1

Андронова-Хопфа - сумма диагональных элементов равна 0;

ОДЗ проверка условий и тд
